# DevInvite ⚡ — Developer Landing Page

---

## Project Overview

DevInvite is a modern and interactive developer landing page designed to showcase collaboration opportunities and invite others to work together. The page features multiple sections including a hero invitation, testimonials, a goals timeline, a collaboration checklist, and contact information. Built with **HTML, CSS, and modern CSS techniques**, this project demonstrates both visual design and frontend development skills.

---

## Features

* Hero section with gradient headings and a glowing line effect
* Call-to-action “Accept the Invitation” button linking to email contact
* Testimonials section with responsive card grid
* Goals timeline showing developer journey and milestones
* Collaboration checklist highlighting key factors for productive teamwork
* Contact section with clickable email link
* Fixed sidebar navigation for smooth page navigation
* Modern dark theme with gradients, shadows, and hover effects
* Fully responsive layout and beginner-friendly code structure
